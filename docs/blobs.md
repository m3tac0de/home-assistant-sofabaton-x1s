# Blobs are now called Command Payloads

The "Blobs" tab of the Sofabaton Control Panel card has been retired, and with
it the "Blob" name. The same functionality — viewing, testing, and saving a
command's stored data — now lives in the **Hub** tab: select a device and
click **Edit device**.

Everything this page used to explain is covered in
[command_payloads.md](./command_payloads.md).
